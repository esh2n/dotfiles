browser (Chrome / Arc) で使用している extension。security、development、navigation、productivity の 4 category。

## Extension 一覧

### Security

| Extension | 説明 |
|-----------|------|
| 1Password | password manager。browser 内での autofill と credential 管理 |

### Development

| Extension | 説明 |
|-----------|------|
| Enhanced GitHub | repo size, file size, download link の表示 |
| JSON Formatter | JSON の format と syntax highlight |
| Material Icons for GitHub | GitHub の file tree に Material Design icon を表示 |
| Refined GitHub | GitHub UI の改善 (200+ features) |

### Navigation

| Extension | 説明 |
|-----------|------|
| Vimium | Vim-style の keyboard navigation。`f` で link hint、`j/k` で scroll |

### Customization

| Extension | 説明 |
|-----------|------|
| Stylus | custom CSS の injection。GitHub や各サイトの外観を調整 |

### Productivity

| Extension | 説明 |
|-----------|------|
| Text Blaze | text expander。template と snippet の展開 |
| Easy Grouping for Google Calendar | calendar の group 切替 |

## 管理

extension list は `~/.config/browsers/extensions.json` で管理している。ID、名前、category、説明を JSON で記録。
